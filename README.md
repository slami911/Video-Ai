## ✨ Overview

**Video-Ai** is a platform for generating videos automatically from text content. Leveraging AI models for script generation, voice synthesis, visual matching, and video composition.

> **Status:** This project is currently in early development. Stay tuned for updates.

## 🚀 Features (Planned)

- [ ] AI script generation from text/articles
- [ ] Multi-language voiceover with ElevenLabs
- [ ] Automatic visual asset matching
- [ ] Custom brand templates and themes
- [ ] Web-based CMS for review and editing
- [ ] Export to MP4 with configurable resolutions
- [ ] Batch processing for multiple videos
- [ ] REST API for programmatic access

## 🛠 Tech Stack (Planned)

| Technology | Purpose |
|-----------|---------|
| **Next.js** | Frontend CMS |
| **FastAPI** | Backend API |
| **Remotion** | Video rendering |
| **OpenAI** | Script generation |
| **ElevenLabs** | Voice synthesis |
| **Docker** | Containerization |
